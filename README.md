## TC-001 — Успешная авторизация валидного пользователя

**Предусловие:**
- Пользователь с логином login1 зарегистрирован в системе
- Пользователь находится на странице авторизации

**Шаги:**
1. Вести Логин: login1
3. Вести Пароль: 123321
4. Нажать на кнопку "Войти"

**ОР:**
- Пользователь успешно авторизован
- Открывается главная страница

**Приоритет:** High
